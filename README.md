# blog
My blog based on [VitePress](https://vitepress.vuejs.org/guide/getting-started).

**Testing**:

```shell
pnpm install
pnpm dev
```

**Building**:

```shell
pnpm build
```

**Deploying**:

See git action in `.github/workflows/deploy.yml`,
it will deploy this project automatically

```shell
# or run ./deploy.sh 
```

write in `blog\zh\blogs`
